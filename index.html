About LearnAI

A self-learning system powered by modern AI — no human teacher required.

⚡

Fast by Design

All AI tasks — summaries, explanations, and quizzes — run on Groq's inference engine using Llama 3.3 70B. This gives you responses in 1–3 seconds instead of 10–15.

🎯

Content-Grounded MCQs

Quiz questions are generated from the actual chapter text, not from the model's general knowledge. Each topic gets 4 unique questions every time.

📖

Smart Chapter Detection

The system first tries pattern matching (Chapter 1, Unit 2, etc.) then falls back to AI detection — so it works on almost any PDF structure.

📊

Progress Tracking

Every quiz result is stored in a local SQLite database. Your scores, chapters, and topics are all tracked so you can see where you're strong and where to improve.

🔄

Fallback Safety

If Groq is unavailable, the system automatically retries with OpenRouter — so you always get a response, just slightly slower.

🛠️

Tech Stack

Flask backend · Groq API (Llama 3.3 70B) · OpenRouter fallback · PyPDF2 extraction · SQLite storage

🔑 API Keys Required
Groq (primary — fast) GROQ_API_KEY Get free key →
OpenRouter (fallback) OPENROUTER_API_KEY Get key →

Set both in your .env file in the project root.
